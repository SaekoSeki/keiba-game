# 🐎 競馬ゲーム（QR参加＋ランキング対応）

ブラウザだけで遊べる6頭立ての競馬（予想・オッズ付き）ゲームです。
今回の拡張で **① QRで参加して全員のランキング表示** と **② 楕円のトラック型コース** に対応しました。

## 何ができる？

- **① QR参加＆ランキング**
  - QRコード（または同じURL）からゲームを開き、名前を登録して参加
  - 各自スマホで自由にレースを遊ぶ
  - レースが終わると所持金がサーバに送られ、**全参加者を「所持金の多い順」でランキング表示**
  - 同じ所持金の人は同順位（例: 1位・2位・2位・4位）
- **② トラック型コース**
  - 直線ではなく **楕円（競馬場風）のコース**を周回
  - カーブで馬の **向きが自然に変わる**

## 動かし方（ローカル）

```bash
# このフォルダ(0623_game_LT)で実行
npm install     # 必要な部品(express, qrcode)を入れる
npm start       # サーバを起動
```

起動したらブラウザで以下を開きます：

```
http://localhost:3000
```

名前の入力を求められるので入力すると参加できます。

## みんなで遊ぶ（同じWi-Fi）

1. サーバを動かしているPCのIPアドレスを調べます。
   - Windows: コマンドプロンプトで `ipconfig` → 「IPv4 アドレス」（例 `192.168.1.5`）
2. 参加者は同じWi-Fiにつないだスマホで `http://192.168.1.5:3000` を開きます。
   - 画面に出ている **QRコード**を読み取ってもOK（同じURLに飛びます）。
3. それぞれ名前を入れて参加 → レースを遊ぶ → ランキングに反映されます。

> メモ: 参加者データは **サーバのメモリ**に置いているだけなので、`npm start` を
> 止める／再起動すると消えます（イベント1回分の割り切り）。残したい場合は
> データベース（Mongoose など）に置き換えてください。

## ファイル構成

| ファイル | 役割 |
|----------|------|
| `index.html` | ゲーム本体（画面・レース・予想・ランキング表示） |
| `app.js`     | Express サーバ（配信＋ 参加/結果/ランキング の API、QR生成） |
| `package.json` | 必要な部品と起動コマンド（`npm start`） |

## サーバの主なAPI

| メソッド/URL | 役割 |
|--------------|------|
| `POST /api/join` | 名前を登録して参加者IDをもらう |
| `POST /api/result` | レース結果（所持金・収支）を送る |
| `GET /api/ranking` | 全参加者を所持金順で並べたランキングを取得 |
| `GET /qr` | このサーバのURLを指すQRコード画像（PNG） |

## サーバ無しでも遊べる

`index.html` をそのままダブルクリックして開いた場合（`file://`）は、
参加・ランキング機能は自動でオフになり、**1人用（ソロ）**として遊べます。
