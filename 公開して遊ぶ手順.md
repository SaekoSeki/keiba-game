# 🌐 自宅からも参加できるようにする手順（案A：トンネル公開）

「各プレイヤーが自宅でQRを読み取って参加 → プレイ → ランキング更新」を実現する手順です。
**あなたのPCで動くサーバに、一時的な公開URL（https）を割り当てる**方法（トンネル）を使います。

```
[ あなたのPC ]                          [ インターネット ]
 npm start (localhost:3000)  ──トンネル──▶  https://xxxx.trycloudflare.com
                                              ▲
                                  自宅の参加者がQRで読み取る
```

> ポイント: **ターミナルを2つ**使います。①サーバ用 と ②トンネル用。両方開けっぱなしにします。

---

## ✅ おすすめ：Cloudflare Tunnel（登録不要・QRがスムーズ）

ngrok無料版は初回に警告ページが挟まり、QRで来た人がつまずきます。
Cloudflareの「クイックトンネル」は**アカウント登録なし・警告なし**で快適です。

### 1. cloudflared をインストール（初回だけ）

PowerShell で：
```powershell
winget install --id Cloudflare.cloudflared
```
インストール後、ターミナルを開き直してください（PATH反映のため）。

### 2. サーバを起動（ターミナル①）

```powershell
cd C:\Users\SaekoSeki\ica_sample\0623_game_LT
npm start
```
`競馬ゲームサーバ起動: http://localhost:3000 …` と出たらOK。**このまま閉じない。**

### 3. トンネルを起動（ターミナル②）

別のターミナルを開いて：
```powershell
cd C:\Users\SaekoSeki\ica_sample\0623_game_LT
npm run tunnel
```
しばらくすると、次のような**公開URL**が表示されます：
```
+--------------------------------------------------------+
|  https://random-words-1234.trycloudflare.com           |
+--------------------------------------------------------+
```
この `https://....trycloudflare.com` が **みんながアクセスするURL** です。**このまま閉じない。**

### 4. ホスト画面を開く → QRを見せる

ホスト用PC（または自分のスマホ）のブラウザで、上の **公開URL** を開きます。
- 画面上部に出る **QRコードは、その公開URLを自動的に指します**（サーバを修正済み）
- 参加者はそのQRを読み取るだけ。自宅でも同じURLに繋がります

### 5. 参加者の流れ

1. QRを読み取る → ゲームが開く
2. 名前を入力して参加
3. レースを遊ぶ → 終了で所持金がサーバへ送信
4. 「🏆 ランキング」に全員の所持金順が反映される

## 🔁 代替：ngrok を使う場合

```powershell
winget install --id Ngrok.Ngrok       # インストール
ngrok config add-authtoken <あなたのトークン>   # 公式サイトで無料登録して取得
npm run tunnel:ngrok                    # = ngrok http 3000
```
表示された `https://xxxx.ngrok-free.app` を開いて使います。
※ 無料版は初回に警告ページ（Visitボタン）が出る点に注意。

## ⚠️ 注意点

- **公開URLは毎回変わります**（クイックトンネルの仕様）。イベントのたびにQR（＝ホスト画面）を出し直せばOK。
- **参加者データはサーバのメモリ管理**です。`npm start` を止める／再起動するとランキングは消えます（イベント1回分）。
  - ずっと残したい・常時公開したい場合は、クラウドにデプロイ＋DB保存（案B）への発展がおすすめです。
- ターミナル①（サーバ）と②（トンネル）は**両方とも開いている間だけ**有効です。
- 会社・学校のネットワークによってはトンネルがブロックされることがあります。その場合はテザリング等の別回線をお試しください。

---

## 困ったとき

| 症状 | 対処 |
|------|------|
| 公開URLを開いても表示されない | ターミナル①（`npm start`）が動いているか確認 |
| QRが古いURLを指す | ホスト画面を**新しい公開URLで開き直す**（QRはそのURLを指す） |
| `winget` が見つからない | Windows の「アプリ インストーラー」を更新、またはストアから導入 |
| ランキングが消えた | サーバを再起動した可能性（メモリ管理のため）。案Bでの常設を検討 |
